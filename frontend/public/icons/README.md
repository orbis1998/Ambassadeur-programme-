# Icônes — à remplir manuellement (ne pas générer automatiquement)

Placez **vos** fichiers ici avec ces noms exacts :

| Fichier | Usage |
|---------|--------|
| `logo-original.png` | Favicon, splash, header, login, inscription |
| `icon-192.png` | Notifications push, écran verrouillé (Apple) |
| `icon-512.png` | Icône app sur l'écran d'accueil mobile (PWA) |

Si vous utilisez `.jpeg`, renommez en `.png` **ou** mettez à jour :
- `src/constants/branding.js`
- `public/manifest.json`
- `public/index.html`
- `public/sw.js`
